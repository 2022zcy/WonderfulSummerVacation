### 任务2 课题摘要

**课题名称：** 基于大规模语言模型的机器人行为生成引擎开发

**摘要：**
本课题旨在开发一个基于大规模语言模型（LLM）的机器人行为生成引擎。任务分为三个阶段：

1. **构建推理引擎：**
   - 研究和选择主流推理引擎（如Transformer、VLLM等），并部署和测试其性能。

2. **优化推理引擎：**
   - 调优推理引擎以最大化其吞吐量，进行基准测试并分析优化效果。

3. **（可选）构建微调训练集：**
   - 基于RoboCodeX框架构建数据集，微调预训练模型，评估其性能和推理精度。
